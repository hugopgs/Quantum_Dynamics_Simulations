# Numerical Simulations of Quantum Many-Body Dynamics in Julia

This repository contains a series of tutorials implementing various numerical methods to simulate the dynamics of the **Transverse Field Ising Model (TFIM)**. The scripts progress from basic exact diagonalization to advanced topics like open quantum systems, mean-field theory, and tensor network methods.

The tutorials are organized by complexity and numerical technique:

### 1. Exact Diagonalization (Closed Systems)
* **`Tuto_1_Dynamics_Transverse_Ising_Hamil.jl`**
    * **Method:** Exact Diagonalization (ED) using dense matrices.
    * **Key Features:** Builds many-body operators using Kronecker products (`kron`) and implements time evolution via direct matrix exponentiation ($U = e^{-iH t}$).
    * **Physics:** Simulates spin-flip dynamics in a 1D Ising chain.

* **`Tuto_2_Spin_model_simulation_Krylov_space.jl`**
    * **Method:** Sparse Matrix ED with Krylov Subspace methods.
    * **Key Features:** Utilizes `SparseArrays` for memory efficiency and `KrylovKit` (Arnoldi method) to approximate the matrix exponential $e^A v$ without fully diagonalizing the Hamiltonian.
    * **Benefit:** Allows simulation of larger systems (e.g., $N=20$) compared to dense ED.

### 2. Open Quantum Systems (Dissipation)
* **`Tuto_3_Density_evolution_in_Master_eq.jl`**
    * **Method:** Lindblad Master Equation (Density Matrix formalism).
    * **Key Features:** Constructs the Liouvillian super-operator $\mathcal{L}$ to simulate decoherence/dissipation ($\gamma$).
    * **Physics:** Tracks the evolution of the density matrix $\rho$ under a non-Hermitian effective Hamiltonian and jump operators.

* **`Tuto_4_quantum_trajectories_ising.jl`**
    * **Method:** Monte Carlo Wave Function (MCWF) / Quantum Trajectories.
    * **Key Features:** A stochastic approach to open systems. It evolves individual wavefunctions using a non-Hermitian Hamiltonian and applies random "quantum jumps" based on decay probabilities.
    * **Comparison:** Averages multiple trajectories ($N_{traj}=500$) to recover the exact Lindblad density matrix result.

### 3. Approximate Methods
* **`Tuto_5_ising_ed_vs_mf_comparison.jl`**
    * **Method:** Mean-Field Theory (MFT) vs. Exact Diagonalization.
    * **Key Features:** Derives the classical equations of motion (Bloch equations) where each spin interacts with a "mean field" generated by its neighbors.
    * **Implementation:** Solves the ODEs using a 4th-order Runge-Kutta (RK4) integrator and compares magnetization dynamics against the exact quantum solution.

### 4. Tensor Networks
* **`Tuto_6_itensors_ghz_trotter.jl`**
    * **Method:** Matrix Product States (MPS) using the **ITensors.jl** library.
    * **Key Features:** * Constructs a GHZ state using a quantum circuit (Hadamard + CNOT cascade).
        * Implements time evolution using Trotter decomposition (Trotter-Suzuki gates).

* **`Tuto_7_mps_vidal_tebd.jl`**
    * **Method:** Custom implementation of Time-Evolving Block Decimation (TEBD) / Vidal's Algorithm.
    * **Key Features:** * Defines a custom `VidalMPS` mutable struct managing $\Gamma$ tensors and $\Lambda$ singular value matrices.
        * Implements the bond-update mechanism using SVD truncation to limit the bond dimension $\chi$.
