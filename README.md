# Quantum Dynamics & Control Simulations

This repository contains a collection of tutorials implementing numerical methods for simulating quantum many-body systems and quantum control protocols. The projects range from basic exact diagonalization to advanced topics like open quantum systems, mean-field theory, and tensor network methods (MPS/TEBD).

The tutorials are organized by physical model and numerical technique:

### 1. Exact Diagonalization (Closed Systems)
* **`Tuto_1_Dynamics_Transverse_Ising_Hamil.jl`**
    * **Method:** Exact Diagonalization (ED) using dense matrices.
    * [cite_start]**Description:** Builds many-body operators using Kronecker products (`kron`) and implements time evolution via direct matrix exponentiation ($U = e^{-iH t}$)[cite: 1, 4, 7].
    * [cite_start]**Physics:** Simulates spin-flip dynamics in a 1D Transverse Field Ising chain[cite: 6].

* **`Tuto_2_Spin_model_simulation_Krylov_space.jl`**
    * **Method:** Sparse Matrix ED with Krylov Subspace methods.
    * **Description:** Utilizes `SparseArrays` and `KrylovKit` (Arnoldi method) to approximate the matrix exponential $e^A v$ without fully diagonalizing the Hamiltonian[cite: 12, 16].
    * [cite_start]**Benefit:** Allows simulation of larger systems (e.g., $N=20$) compared to dense ED[cite: 20].

### 2. Open Quantum Systems (Dissipation)
* **`Tuto_3_Density_evolution_in_Master_eq.jl`**
    * **Method:** Lindblad Master Equation (Density Matrix formalism).
    * [cite_start]**Description:** Constructs the Liouvillian super-operator $\mathcal{L}$ to simulate decoherence and dissipation ($\gamma$)[cite: 28, 29].
    * **Physics:** Tracks the evolution of the full density matrix $\rho$ under a non-Hermitian effective Hamiltonian[cite: 30].

* **`Tuto_4_quantum_trajectories_ising.jl`**
    * **Method:** Monte Carlo Wave Function (MCWF) / Quantum Trajectories.
    * **Description:** A stochastic approach to open systems. [cite_start]It evolves individual wavefunctions using a non-Hermitian Hamiltonian and applies random "quantum jumps" based on decay probabilities[cite: 39, 43, 44].
    * [cite_start]**Comparison:** Averages multiple trajectories ($N_{traj}=500$) to recover the exact Lindblad density matrix result[cite: 49, 50].

### 3. Approximate Methods & Comparisons
* **`Tuto_5_ising_ed_vs_mf_comparison.jl`**
    * **Method:** Mean-Field Theory (MFT) vs. Exact Diagonalization.
    * [cite_start]**Description:** Derives the classical equations of motion (Bloch equations) where each spin interacts with a "mean field" generated by its neighbors[cite: 56, 57].
    * **Implementation:** Solves the ODEs using a 4th-order Runge-Kutta (RK4) integrator and compares magnetization dynamics against the exact quantum solution[cite: 58, 61].

### 4. Tensor Networks (MPS)
* **`Tuto_6_itensors_ghz_trotter.jl`**
    * **Method:** Matrix Product States (MPS) using the **ITensors.jl** library.
    * [cite_start]**Description:** Constructs a GHZ state using a quantum circuit and implements time evolution using Trotter decomposition (Trotter-Suzuki gates)[cite: 62, 64].

* **`Tuto_7_mps_vidal_tebd.jl`**
    * **Method:** Custom implementation of Time-Evolving Block Decimation (TEBD) / Vidal's Algorithm.
    * [cite_start]**Description:** Defines a custom `VidalMPS` struct managing $\Gamma$ tensors and $\Lambda$ singular value matrices[cite: 70]. [cite_start]Implements bond updates using SVD truncation to limit the bond dimension $\chi$[cite: 74, 75].

### 5. Quantum Control (Python)
* **`ramsey_fringes.py`**
    * **Method:** Analytical Time Evolution (Python).
    * **Description:** Simulates **Ramsey Interferometry** sequences. Calculates the excitation probability $P_e(\Delta)$ as a function of detuning and pulse duration to visualize Ramsey fringes and the effect of pulse area ($\pi/2$ vs $\pi$ pulses).
